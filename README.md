# Personal Portfolio

This repository contains a simple personal portfolio website built with vanilla HTML and CSS by a two–student team.

## Features

- Responsive layout for mobile, tablet, and desktop.
- Sections: Home, About, Projects, Contact.
- Clean, accessible UI with semantic HTML.
- Simple, reusable CSS components (buttons, cards, navigation).

## Technologies

- HTML5
- CSS3 (no framework, no JavaScript)

## Getting Started

1. Clone the repository:
   git clone https://github.com/USERNAME/REPO-NAME.git

3. Open `index.html` directly in a browser  
or use a simple local server (for example with VS Code Live Server).

No build step or installation is required.
